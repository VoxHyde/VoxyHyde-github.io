Clear the pyramid by removing exposed cards in pairs that total 13. Kings clear alone.

The waste pile is optional. You can still clear two exposed pyramid cards together without using it.

Draws come from the remaining deck, and when the stock runs out you get one redeal from the waste pile.

New deals now start from a real shuffled 52-card deck, then bad or too-free boards get filtered out before the run begins.

Local Leaderboard

Best runs on this device, ranked by the fewest moves.

Card Values

A = 1, J = 11, Q = 12, K = 13.

Ace does not count as 10 in this version.

Any two exposed cards that add to 13 can be cleared together.

Quick Tips

Try to clear cards from the lower rows first so you uncover more possible pairs.

If a card is covering another, it cannot be selected yet.

The deal still has randomness, but it should now feel like a real deck instead of an over-scripted free win.

How to Play

  • Click any exposed pyramid card to select it.
  • Select another exposed card so the two values add up to 13.
  • Kings are worth 13, so they clear by themselves.
  • Aces are worth 1 here, not 10.
  • Use `Draw` to flip the next stock card into the waste pile.
  • Each new deal starts from a normal 52-card shuffle and gets filtered so the run has a real winning path.
  • Clear the whole pyramid to win, then save your name to the local leaderboard.
